🚀 Agentic AI Deep-Dive
This repository contains a series of implementations focused on Agentic AI design patterns. As a technical professional in the cybersecurity space, I’ve built these to explore how autonomous agents can solve complex, stateful problems.

🛠️ Key Projects
Multi-Agent Orchestration: A pipeline coordinating Researcher, Designer, and Copywriter agents using GPT-4o and DALL-E 3.

Reflection & Self-Correction: A self-improving chart generation workflow that critques its own code via vision-capable LLMs.

Code-as-Action: An operational agent that writes and executes Python/TinyDB code to manage database transactions.

Automated Evaluation: A component-level evaluation system to verify AI source quality.

🧪 Technical Stack
Languages: Python (Pandas, Matplotlib, TinyDB)

AI Frameworks: OpenAI API, AISuite, Anthropic Claude

Patterns: Multi-agent systems, Reflection, Tool-calling, Multimodal Vision
